Procedure about moving to Ikoma

オンライン手続き一覧

Moving online procedure list

電気

関西電力

Kansai Electric Power

電気の停止・開始の手続きをインターネットから行うことができます。開始・停止を同時に行うことができるので便利です。
https://kepco.jp/service/move

水道

生駒市水道局

Ikoma City Water Utility

生駒市で水道を使用する際には、生駒市への届け出が必要です。開始・中止について、奈良県の「自治体共同運営システム（e古都なら）」を使って、手続きすることが可能です。「e古都なら」で、水道の開栓・閉栓をそれぞれ選択して予約してください。
https://s-kantan.jp/ikoma-nara-u/offer/offerList_initDisplay.action

ガス・電気

大阪ガス

Osaka Gas

生駒市のガスは、プロパンガスでなければ大阪ガスに使用開始・使用中止の連絡が必要です。大阪ガスで電気を申し込むこともできます。
http://home.osakagas.co.jp/contact/removal/index.html

郵便

郵便局

Post office

e転居サービスで郵便局に申請しておくと、引っ越し後一年間、郵便物を転送してくれます。
https://welcometown.post.japanpost.jp/etn/index.html

宅急便

ヤマト運輸

kuroneko yamato

オンラインでの申請で、ヤマト運輸の宅急便の転送が可能になります。有効期間は1年間です。
http://www.kuronekoyamato.co.jp/ytc/customer/members/receive/tenkyo/

http://www.police.pref.nara.jp/0000000424.html

About CODE for IKOMA

CODE for IKOMA

生駒の地域課題解決や魅力発信をITで

CODE for IKOMA（コード・フォー・イコマ）は、奈良県生駒市を中心に、ITを活用した地域課題の解決や地域の活性化を行っているコミュニティです。 生駒市の市民活動団体として、市主催のITイベントのアドバイザーとしても活動しています。
